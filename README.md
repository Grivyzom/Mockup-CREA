# Introducción

### Descripción General

Portal web desarrollado en **Angular 20** con **Tailwind CSS** para la gestión y visualización de proyectos estudiantiles en INACAP. La aplicación permite a estudiantes y docentes explorar, crear y gestionar proyectos interdisciplinarios con un enfoque en "Aprender Haciendo".
